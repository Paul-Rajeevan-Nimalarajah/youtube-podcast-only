# YouTube Podcast Only

A Chrome extension that filters the YouTube Home feed to show only long-form podcast-style content.

## Features
- Hides music, shorts, and clickbait
- Shows podcasts, interviews, and long discussions
- Runs fully locally (no data collection)
- Lightweight and fast

## How it works
The extension analyzes video title text and duration on the YouTube Home feed and hides non-podcast content.

## Limitations
- Does not recommend new videos
- Works only on videos YouTube already loads

## Installation (Local)
1. Download the folder
2. Open chrome://extensions
3. Enable Developer mode
4. Click "Load unpacked"
5. Select the folder
